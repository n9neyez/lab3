Declarative Rendering

Vue at the core, enables users to declaratively render data to the DOM (Document Object Model) using template syntax.


{{message}}

Here, Vue can call the text "What's Up?" from within a data object in a Vue instance. The data and DOM are linked now and is reactive. We can test this using our browser's JavaScript console; setting app.message = "any text" will change the the above text accordingly.

We can also bind element attributes with a directive: v-bind. Directives are always prefixed with -v to show they are provided by Vue, these directives apply reactive behavior to the rendered DOM.


Hover over this text to see how Vue interacts with element attributes.

The span element's title attribute is kept up-to-date with the message property on the Vue instance (app2). Opening the console, we can enter app2.message = "some text..." and we can see the bound HTML, the title attribute will be updated.
